# Task Management Backend

This is a Node.js Express backend for a task management app. It will use PostgreSQL via Prisma ORM. 

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm run dev
   ```
